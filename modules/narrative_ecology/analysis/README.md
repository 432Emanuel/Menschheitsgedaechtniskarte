# Analyse-Modus (Narrativökologie)

## Perspektive

Vogelperspektive / Abstand – Strukturbeobachtung von Narrativen als ökologische Systeme.

## Fokus

- Identifikation archetypischer Narrative
- Analyse von Feedback-Loops und Selbstverstärkung
- Kartografie von Macht-Mechanismen
- Verfolgung von Institutionalisierungspfaden
- Beobachtung von Dominanz-Zyklen

## Prinzipien

- Beobachtend, nicht bewertend
- Strukturelle Muster erkennen
- Ökologische Metapher: Konkurrenz, Hybridisierung, Invasion, Verdrängung
- Klarheit über das, was wirkt

## Dateien

1. **narrative_archetypes.json** – Archetypische Narrativformen als neutrale Muster
2. **feedback_loops.json** – Selbstverstärkende Kreisläufe
3. **power_mechanisms.json** – Mechanismen der Narrativ-Stärkung
4. **institutionalization_paths.json** – Pfade zur Institutionalisierung
5. **dominance_cycles.json** – Ökologische Zyklen der Narrativ-Konkurrenz

## Status

**Draft** – Alle Inhalte sind Strukturgerüste für weitere Entwicklung.
