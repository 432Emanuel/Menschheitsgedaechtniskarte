# Narrativökologie (NE-001)

## Definition

Narrativökologie ist ein strukturbeobachtendes Modul zur Analyse von Narrativen als ökologische Systeme. Sie untersucht, wie Geschichten entstehen, sich verbreiten, miteinander konkurrieren und kollektive Realität formen – ohne normative Bewertung.

## Zwei Modi

### Analyse (Vogelperspektive / Abstand)
- Strukturelle Beobachtung von Narrativen
- Identifikation von Mustern, Feedback-Loops und Macht-Mechanismen
- Kartografie von Narrativ-Landschaften
- Ziel: Klarheit über das, was wirkt

### Erfahrungsraum (Innenperspektive / Resonanz)
- Persönliche Auseinandersetzung mit eigenen Narrativen
- Spiegelung von Konflikten und Resonanzen
- Experimente zum Perspektivwechsel
- Ziel: Klarheit über das, was bewegt

## Kernprinzipien

- **Wegweiser, nicht Weggeber**: Die Richtung ist „Klarheit“, das Ergebnis bleibt dem Nutzer überlassen
- **Strukturbeobachtend**: Keine Bewertung, keine politische Agenda, keine Missionierung
- **Ökologische Metapher**: Myzel = Querverbindungen, Pfade = Übergänge, Lichtungen = Erkenntnismomente
- **Dualität**: Analyse und Erfahrungsraum ergänzen sich – beide Perspektiven sind notwendig
