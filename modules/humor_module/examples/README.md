# Beispiele

Dieser Ordner enthält optionale Beispiele zur Illustration des Humor-Moduls.

## Zweck

Beispiele zeigen mögliche Formate und Vorgehensweisen, sind aber nicht verbindlich. Sie dienen als Inspiration, nicht als Vorlage.

## Hinweis

Dieser Ordner kann leer sein. Beispiele werden nach Ermessen des Moduls hinzugefügt, wenn das Modul sie als nützlich erachtet.
