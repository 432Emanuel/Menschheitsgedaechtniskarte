# Build on Old - Schichtungs-Analyse

Dieses Modul analysiert und dokumentiert, wie neue Strukturen (materiell, semantisch, praktisch, institutionell) auf vorhandenen Fundamenten aufsetzen statt bei Null zu beginnen.

## Kernkonzept

"Weiterbauen auf Altem" ist ein universelles anthropologisches Prinzip:
- Menschen und Kulturen setzen neue Formen auf bestehende Grundlagen auf
- Das Alte wird genutzt, umgedeutet, verstärkt oder überformt
- Nicht durch Bruch/Ersetzung, sondern durch Kontinuität und Adaption

## Typen des Weiterbauens

### 1. Materielles Weiterbauen (BOO_MATERIAL)
Neubauten, Erweiterungen oder Umnutzungen auf älteren Fundamenten, Ruinen, Terrassen, Mauern, Trümmern.

**Signals:**
- Schicht-Datierungen weichen ab (Fundament/Unterbau deutlich älter oder jünger als Aufbau)
- Spolien / wiederverwendetes Material
- Stilmix / Traditionsmix
- Kontextlücke (Form bleibt sichtbar, ursprüngliche Funktion nicht mehr rekonstruierbar)
- Versionierungsspur (Spuren von Reparatur, Ausbau, Umwidmung)

**Examples:**
- Römische Tempel auf griechischen Fundamenten
- Kirchen auf heidnischen Kultstätten
- Moderne Gebäude in historischer Bausubstanz

### 2. Semantisches Weiterbauen (BOO_SEMANTIC)
Alte Symbole, Begriffe, Rituale oder Narrative werden beibehalten, aber neu gerahmt und anders gedeutet.

**Signals:**
- Symbol-Recycling
- Bedeutungsverschiebung
- Terminologie-Übernahme
- Narrative-Adaption

**Examples:**
- Christliche Feiertage auf heidnischen Festen
- Politische Slogans auf alten Mythen
- Wissenschaftliche Begriffe im allgemeinen Sprachgebrauch

### 3. Praktisches Weiterbauen (BOO_PRACTICE)
Verbesserung von Routinen/Techniken über Iteration: Waffen, Werkzeuge, Jagdmethoden, Handwerk, wissenschaftliche Verfahren.

**Signals:**
- Iterative Verbesserung
- Prototypen-Serien
- Fehlerkultur
- Knowledge-Cumulation

**Examples:**
- Werkzeug-Entwicklung (Stein → Bronze → Eisen)
- Landwirtschaftliche Methoden
- Wissenschaftliche Methodik

### 4. Institutionelles Weiterbauen (BOO_INSTITUTION)
Neue Ordnungssysteme setzen auf bestehenden Organisationsformen auf: Recht, Verwaltung, Bildung, Imperien, Kirchen.

**Signals:**
- Institutionelle Kontinuität
- Bürokratische Schichtung
- Rollen-Adaption
- Governance-Iteration

**Examples:**
- Römisches Recht in modernen Rechtssystemen
- Bürokratische Strukturen über Epochen hinweg
- Bildungssysteme auf antiken Modellen

## Overbuild & Spolien

### Overbuild
Neue Struktur wird direkt über ältere Struktur gebaut, ohne diese zu entfernen:
- Physischer Overbuild: Gebäude über Gebäuden
- Semantischer Overbuild: Neue Bedeutung über alter Symbolik
- Institutioneller Overbuild: Neue Institution auf alter Struktur

### Spolien
Wiederverwendete Teile aus älteren Kontexten:
- Materielle Spolien: Architekturteile, Inschriften, Säulen
- Semantische Spolien: Begriffe, Konzepte, Narrative
- Institutionelle Spolien: Organisationsformen, Rollen, Prozesse

## Iteration statt Neuanfang

Wichtiges Prinzip:
- Nicht "von vorne beginnen", sondern "auf existierendem aufbauen"
- Fehlerkultur statt Perfektionismus
- Versionierung statt Finalität

### Reifegrade
- **v0**: Rudimentär (spontan/unbewusst)
- **v1**: Bewusst (wiederholbar, benennbar)
- **v2**: Integriert (stabil, kontextübertragbar)
- **v3**: Generativ (kreativ erweiterbar, lehrbar)

## Häufige Missverständnisse

### Fehlinterpretation: Alles muss aus einer Quelle stammen
Überlagerung wird fälschlich als einheitliche Planung interpretiert.

### Fehlinterpretation: Ausrichtung = astronomische Funktion
Orientierung kann auch ästhetisch, prestigebezogen, rituell oder zufällig-kontextuell sein.

### Fehlinterpretation: Imperien als Ursprung
Das Prinzip ist basal (Lernen, Nutzen, Kontinuität) und geht Imperien oft voraus.
