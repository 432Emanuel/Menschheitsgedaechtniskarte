# Time Witness

## Zweck

Dieser Basket sammelt subjektive Berichte und Augenzeugenberichte zur COVID-19-Pandemie im Jahr 2020.

## Inhalt

- Persönliche Interviews
- Zeitzeugen-Aufzeichnungen
- Transkripte von mündlichen Berichten
- Tagebucheinträge (ohne Interpretation)

## Prinzipien

- Keine Bewertungen oder Interpretationen
- Nur Rohmaterial in originaler Form
- Neutralität bewahren
