# Structure Markers

## Zweck

Dieser Basket sammelt strukturelle Merkmale und systemische Beobachtungen zur COVID-19-Pandemie im Jahr 2020.

## Inhalt

- Verwaltungsprozesse und Entscheidungswege
- Mechanismen der Krisenbewältigung
- Systemdynamiken und institutionelle Abläufe
- Organisatorische Muster

## Prinzipien

- Fokus auf strukturelle Aspekte
- Keine normativen Bewertungen
- Dokumentation von Prozessen und Mechanismen
- Neutralität bewahren
