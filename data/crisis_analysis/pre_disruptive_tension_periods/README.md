## Five Tension Indicators

1. **Discourse Density**
   - Documented public narratives (end-time discourses, crisis narratives)
   - Media production rates
   - Publication frequency

2. **Technological Velocity**
   - Rate of innovation adoption
   - Infrastructure transformation
   - Communication technology shifts

3. **Economic Instability**
   - Market volatility
   - Financial crises
   - Structural economic shifts

4. **Polarization Degree**
   - Political fragmentation
   - Social cleavages
   - Ideological antagonism

5. **Symbolic Overproduction**
   - Artistic/cultural production spikes
   - New ideological movements
   - Symbol system diversification

## Four-Level Structure

Each time block separates:

1. **Data Level:** Verifiable facts, documented events, statistics
2. **Discourse Level:** Public narratives, media discourse, published theories
3. **Structure Level:** Aggregate tension assessment, pattern identification
4. **Interpretation Level:** Explicitly marked hypotheses (always speculative)
