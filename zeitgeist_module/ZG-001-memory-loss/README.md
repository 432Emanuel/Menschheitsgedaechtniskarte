# ZG-001: Erinnerungsverlust & Ersatzrituale

## Zweck

Dieses Zeitgeist-Modul beschreibt gesellschaftliche Situationen, in denen direkte existenzielle Erfahrung (z.B. Krieg, Hunger, Naturabhängigkeit) verloren geht und durch Ersatzrituale, Narrative oder mediale Simulationen kompensiert wird.

## Grundspannung

- Erfahrung ↔ Abstraktion
- Erinnerung ↔ Simulation

## Nutzung in der Menschheitsgedächtniskarte

Das Modul dient als analytische Linse für das Verständnis archaischer Orte. Es hilft:

1. **Orte als Interfaces zu lesen**: Nicht als religiöse oder ästhetische Monumente, sondern als funktionale Gedächtnisstrukturen, die Erfahrungsräume stabilisieren
2. **Universelle Muster zu erkennen**: Parallelen zwischen neolithischen Gedächtnisstrukturen und heutigen Ersatzmechanismen
3. **Perspektivwechsel zu ermöglichen**: Von romantischer Verklärung zu funktionaler Analyse

## Struktur des Moduls

- `module_meta.json`: Strukturierte Metadaten für maschinelle Weiterverarbeitung
- `modern_context.md`: Analyse aktueller Phänomene des Erinnerungsverlusts
- `ancient_mappings/goseck.md`: Fallstudie zur Kreisgrabenanlage Goseck
- `perspective_exercises.md`: Konkrete Übungen für Nutzer der Karte
- `cross_cultural_notes.md`: Hinweise zur Übertragbarkeit auf andere Kulturräume

## Kategorisierung

- **Kategorie**: Kollektives Bewusstsein
- **Modul-ID**: ZG-001-memory-loss
- **Zeitlosigkeit**: Dieses Modul beschreibt kein historisches Phänomen, sondern ein psychologisches und soziologisches Muster, das in verschiedenen Epochen auftritt.
